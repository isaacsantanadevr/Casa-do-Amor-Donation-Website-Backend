# Casa-do-Amor-Donation-Website-Backend
Neste repositório, será desenvolvida a parte backend de um projeto filantrópico, no formato de um website de doações voltados a arrecadar fundos e receber inscrições de voluntários para a Casa do Amor, instituição de tratamento oncológico de Vitória da Conquista - BA. Projeto para as disciplinas: Engenharia de Software e Banco de Dados I
